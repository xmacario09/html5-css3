Trabalhando com listas

Listas Ordenadas (Listas que a ordem faz diferença)

Utilizando a tag OL para criar o campo da lista e dentro dele a tag LI para listar os ítens.

Não é obrigatório o /LI para fechar a listagem, a partir do HTML 5.

  1. Acordar
  2. Escovar os dentes
  3. Tomar café
  4. Treinar
  5. Ler a Blíblia
  6. Trabalho

Listas não ordenadas (Listas que não tem necessidade de uma ordem)

Utilizando a tag UL para criar a lista não ordenada, após isso, seguindo o mesmo raciocínio da tag LI.

  • Comprar feijão
  • Comprar Arroz
  • Compar Pão
  • Compar leite
  • Compar queijo
